No CSS nós possuimos os Seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag Html
  • Seletor por ID (#): Busca elementos através do atributos "id"
  • Seletr por Chave(.): Busca elemetos através do atributo "class"
  • Seletor Universal(*): Seleciona todos os elementos HTML
  • Seletor de Atríbutos ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atríbutos com um valor específico
